    BackGround - Imagens de fundo

    Ajustando imagens de fundo usando :
    background-size
    background-repeat

      background-size: auto; background-repeat: repeat;

      Size> esse atributo irá redimensionar a imagem de forma automática e ele é selecionado por padrão no CSS, fazendo com que a imagem de fundo mantenha a qualidade ideal para o seu tamanho

      Repeat: esse valor é definido por padrão no css e ele vai fazer com que a imagem se repita para caber dentro do tamanho da área caso a imagem seja menor do que o tamanho da area

      background-size: cover;

      Usando o atributo cover, a imagem irá cobrir todo a área de fundo, embora ela seja grande demais ou pequena para aquele tamanho
      background-size: contain; background-repeat: repeat-x;

      Size Esse valor vai redimensionar a imagem fazendo com que ela apareça completamente dentro da área, embora seja necessária a repetição da imagem.

      Repeat-x Esse valor irá definir o x para a repetição da imagem.

      background-size: valor; background-repeat: repeat-y;

      Size:Também podemos definir o tamanho da nossa imagem, passando os valores em pixels, em porcentagem
      passando somente um valor, estamos passando somente o parametro para largura da nossa imagem, e então a altura será definida automaticamente.

      repeat-y: esse valor irá fazer com que a imagem se repita somente no eixo vertical.

      background-size: valor L, valor A; background-repeat: space;

      Passando o valor dos eixos X e Y, podemos definir largura e altura da imagem, podendo assim, manipulá-la de maneira mais livre.

      repeat-space: esse valor fará com que a imagem não seja cortada e se repita conforme necessária para preenchimento da área, caso a imagem precise ser cortada, o space criará uma lacuna em branco para evitar o corte da imagem.
      background-size: camadas;

      Também podemos estilizar as camadas, alterando o seu tamanho
      background-repeat: round;

      Com o valor round a imagem se repete tocando nas quatro bordas da area pré definida mas ele não corta a imagem, com isso a imagem está sendo redimensionada automaticamente
      background-repeat: ;

      Com esse valor, a imagem não irá se repetir

      background-repeat: eixo x e eixo y;

      Podemos passar valores para o repeat nos eixos X e Y manipulando as repetições
